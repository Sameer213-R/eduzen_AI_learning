# 🎓 EduZen – AI Learning Platform

> **An AI-powered personalized learning platform built with Python, Flask, Google Gemini, and Jinja2 to make learning interactive and engaging for students.**

## 📌 The Problem

Traditional online learning platforms often provide the same content to every student, regardless of their age or learning level.

* Learning is not personalized.
* Students have limited interaction while studying.
* Teachers spend significant time creating quizzes and learning materials.
* Young children need safe, age-appropriate AI assistance.

## 💡 The Solution

**EduZen** delivers a personalized learning experience where students can learn based on their **grade and age**. The platform uses AI to answer questions, generate quizzes, create audio lessons, and provide interactive learning tailored to each student's level.

## ✨ Key Features

### 👨‍🎓 For Students

* Learn according to grade and age
* Chat with an AI learning assistant
* Get age-appropriate answers
* Generate quizzes instantly
* Generate audio lessons
* Interactive learning experience

### 👩‍🏫 For Teachers

* Generate quizzes using AI
* Create learning content faster
* Save time preparing assessments
